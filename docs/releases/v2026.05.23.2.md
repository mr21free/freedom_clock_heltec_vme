# Freedom Clock 2026.05.23.2

## Fixes

- **Static wealth screen ghosting** - switching screens in static wealth mode no longer leaves the previous screen visible as a ghost image. The previous screen was showing through because fast (partial) refresh was used without a base frame to clear from. Static wealth screen changes now use a full refresh, which clears the panel cleanly before drawing the new screen.

## Builder Notes

```bash
./tools/FreedomClockSecurityTool.command build-manual-update --release-name freedom-clock-v2026.05.23.2 --board e213
./tools/FreedomClockSecurityTool.command build-manual-update --release-name freedom-clock-v2026.05.23.2 --board e290
```

Use the `.bin` file matching the physical board model. Do not flash an E213 build onto E290 hardware, or the other way around.
