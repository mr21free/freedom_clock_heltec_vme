# Freedom Clock 2026.05.05.8

## Highlights

- Reworked the setup-page update links for iPhone captive portal mode.
- GitHub release URLs are now shown as copyable text instead of relying on external navigation from the captive Wi-Fi browser.
- Added copy buttons for the main GitHub Releases URL and the latest release URL shown after checking for updates.

## Why This Changed

On iOS, the captive Wi-Fi browser cannot reliably open external GitHub pages while connected to the Freedom Clock setup Wi-Fi. Showing copyable URLs makes the flow more honest: users can copy or manually select the release URL, then open it later in Safari or another browser with normal internet access.

## Update Packages

- `FreedomClock-2026.05.05.8-manual-update-open.bin` for normal devices
- `FreedomClock-2026.05.05.8-manual-update-secure.bin` for security-hardened devices

## Notes

- Saved settings stay on the device during normal firmware updates.
- The setup-page release checker still shows release notes directly on the local setup page when the device has working Wi-Fi internet access.
