# Freedom Clock 2026.05.05.7

## Highlights

- Added a `Check Latest Release` button to the local setup page so the device can fetch the newest published GitHub Release and show the release notes directly in the browser.
- Improved the firmware-update section of the setup page so it is visually separate from validation and easier to understand.
- Kept the local manual firmware upload flow simple: users can still upload the `.bin` file directly from the setup page and keep their saved settings.

## Screen Updates

- Screen 2 now focuses on actual input parameters and firmware information.
- Removed `DISPLAY MODE` from Screen 2.
- Confirmed the 4-screen model:
  - Screen 1: expected freedom time, expected lifetime, freedom coverage
  - Screen 2: input parameters + firmware
  - Screen 3: current wealth
  - Screen 4: freedom change

## Update Packages

- `FreedomClock-2026.05.05.7-manual-update-open.bin` for normal devices
- `FreedomClock-2026.05.05.7-manual-update-secure.bin` for security-hardened devices

## Notes

- The setup-page release checker only works against published GitHub Releases, not plain tags.
- Saved settings stay on the device during normal firmware updates.
