# Freedom Clock 2026.05.26.1

## New functionality

- Added optional monthly income and income growth to the freedom-time model.
- Monthly income defaults to `0`, so existing calculations are unchanged unless the user enables it.
- Added monthly income to the setup page, preview calculation, settings screen, and device-side freedom calculations.
- Added support for 1-minute refresh intervals for accelerated testing and battery calibration workflows.
- Expanded battery stats logging to store up to 1440 samples.

## Fixes and reliability

- Fixed E213 setup screen layout so the Wi-Fi name no longer overlaps the QR code.
- Improved Wi-Fi connection reliability with retries, Wi-Fi sleep disabled during connection, and richer diagnostics.
- Fixed refresh interval validation so invalid values no longer silently clamp while still showing settings saved.
- Fixed iPhone decimal input handling by accepting comma and dot decimal formats where needed.
- Fixed freedom coverage rounding so the device matches the website calculation more closely.
- Cleaned up factory-reset restart state so reset and welcome flows are less likely to inherit stale setup state.

## Display and UX polish

- Improved quote screen typography so medium-length quotes can use larger text when they fit.
- Added at least 1px line spacing for larger quote text to prevent lines from touching.
- Rebalanced quote author placement relative to the footer separator.
- Moved the battery icon and battery percentage 2px upward on device screens.
- Added monthly income to the settings screen.

## Installation

- `FreedomClock-2026.05.26.1-E213-manual-update-open.bin` for normal E213 devices
- `FreedomClock-2026.05.26.1-E290-manual-update-open.bin` for normal E290 devices
- `FreedomClock-2026.05.26.1-E213-manual-update-secure.bin` for security-hardened E213 devices
- `FreedomClock-2026.05.26.1-E290-manual-update-secure.bin` for security-hardened E290 devices
