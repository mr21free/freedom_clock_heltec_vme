# Freedom Clock 2026.05.05.9

## Highlights

- Added direct online firmware installation from the setup page.
- Manual `.bin` upload remains available as the fallback update path.
- The device now selects the right release asset automatically:
  - `open` package for normal devices
  - `secure` package for security-hardened devices

## Firmware Update Flow

- `Check Latest Release` still loads the newest GitHub Release notes directly on the setup page.
- If the published release is newer than the installed firmware, `Update Firmware` becomes available.
- The device downloads the matching `.bin` package over Wi-Fi, writes it to flash, and reboots.
- Online install refuses to downgrade to an older published release. Manual upload can still be used for deliberate recovery or testing.

## Notes

- This needs working Wi-Fi with internet access while setup mode is open.
- Saved settings stay on the device during normal firmware updates.
- This version is intended for local testing first. Publish to GitHub Releases only after explicit approval.
