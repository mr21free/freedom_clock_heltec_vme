# Freedom Clock 2026.05.06.1

## Highlights

- Added a Freedom Time Preview to the setup page so users can see the expected `Y / M / W` result before saving.
- The preview mirrors the firmware freedom-time model, including the `FOREVER` cap.
- Added a `SETTINGS` title to screen 2 and aligned title placement across screens 2, 3, and 4.
- Added an optional front-light hardware guide for night reading.

## Setup Preview

- Static net worth previews immediately from the entered wealth amount.
- Static BTC previews after `Test Connection` fetches a BTC/USD price.
- BTC via MQTT previews after `Test Connection` receives both BTC price and BTC amount.

## Hardware Notes

- The front-light guide recommends a warm-white e-ink front light, not a display backlight.
- First prototype recommendation is a physical momentary button with low-current warm-white LEDs.
- `GPIO42` is reserved as the clean optional firmware-control pin if the hardware layout is later tested.

## Release Rule

- Build and test locally first.
- Publish to GitHub Releases only after explicit approval.
