# Freedom Clock 2026.05.11.1

## Highlights

- Daily Quote: a new optional screen that shows a motivational quote at each refresh instead of the main clock, cycling through the full quote library before repeating.
- Auto-update: the device can now check for and install new software automatically at each boot. Off by default.
- Life beyond expectancy: if the current date exceeds the configured life expectancy, the device shows the bonus time with a `+` prefix, relabels the section "BEYOND LIFE EXPECTANCY", and replaces freedom coverage with N/A.

## New Features

- **Daily Quote screen** — enabled in setup under "Daily Quote". At each refresh a motivational quote is selected and shown with freedom time, lifetime left, and coverage at the bottom. The device cycles through all quotes before repeating.
- **Auto-update** — toggle in setup under "Software Update". When on, the device checks GitHub at each boot and installs the latest release automatically if one is available.
- **Scheduled daily wake** — when the refresh interval is set to 1 day, a time picker appears to set the exact UTC wake time. The device wakes at that time each day instead of counting from the previous sleep.
- **Life beyond expectancy** — age past the configured life expectancy now shows bonus time with a `+` prefix (e.g. `+2y 5m 3w`). Freedom coverage shows N/A on the main screen and is omitted on the quote screen since the metric no longer applies.
- **Time format** — new setting to choose between Years / Months / Weeks or Weeks-only display across all screens.
- **Battery percentage** — new setting to show or hide the percentage number next to the battery icon. Off by default; when off the icon moves to the far right of the screen.

## Fixes

- **Setup Wi-Fi password is now stable** — the portal password is derived from the device ID and stays the same across reboots and factory resets. Previously it was randomized each time setup mode started. The password is shown on the e-ink screen.
- **Wi-Fi scan on page load** — the setup page now scans nearby networks automatically when it opens instead of showing a static "Loading nearby networks…" placeholder that was never replaced.
- **Welcome screen after factory reset** — fixed garbled display after factory reset. The Heltec e-ink library's partial-refresh window state persisted after the ellipsis animation on the reset screen, causing the next full-screen draw to only update a small region. Fixed by resetting the window to full-screen before each screen draw.

## Setup Page

- **Validate button removed** — validation now runs automatically as part of the save flow. The Save button triggers a connection check followed by save, with inline feedback throughout. No separate step required.
- **Software Update** — the update section is renamed from "Firmware Update". Automatic and manual update modes are shown or hidden based on the auto-update setting. Checking for updates is wired to the auto-update toggle.

## Builder Notes

```bash
./tools/FreedomClockSecurityTool.command build-manual-update --release-name freedom-clock-v2026.05.11.1 --board e213
./tools/FreedomClockSecurityTool.command build-manual-update --release-name freedom-clock-v2026.05.11.1 --board e290
```

Use the `.bin` file matching the physical board model. Do not flash an E213 build onto E290 hardware, or the other way around.
